# Overview

Find the location(State and City) of a zipcode.

### MVP
- Signup / Login

### Requirements

*Must have 5 componets
1. Home
2. NavBar
3. Signup
4. Login
5. Enter Zipcode
6. Zipcode Card

*Must have 3 client-side-routes
1. / = Home
2. /signup = Signup
3. /login = Login
4. /enterZipcode = Find location of a zipcode
